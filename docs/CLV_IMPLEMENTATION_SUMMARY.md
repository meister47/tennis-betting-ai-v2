## 🔄 РАБОЧИЙ ПРОЦЕСС

### РЕЖИМ С CLV (CLV_MODE_ENABLED=true)
```
1. Анализ матчей → Рекомендации
2. Создание ставки → status: 'waiting_for_closing_line'
3. Сбор closing коэффициентов (каждые 10 минут)
4. Решение:
   - Линия упала/стабильна → status: 'pending' ✅
   - Линия выросла → status: 'skipped_line_movement' ❌
5. Ставка выполняется/пропускается
6. Анализ эффективности CLV
```

### РЕЖИМ БЕЗ CLV (CLV_MODE_ENABLED=false)
```
1. Анализ матчей → Рекомендации
2. Создание ставки → status: 'pending'
3. Ставка выполняется сразу
```

## 📊 ПОКАЗАТЕЛИ КАЧЕСТВА

### ОЖИДАЕМЫЕ РЕЗУЛЬТАТЫ
1. **Win Rate положительного CLV** > 55%
2. **Win Rate отрицательного CLV** < 45%
3. **Разница** > 10% в пользу положительного CLV
4. **Общий ROI** увеличение на 3-5%

### МЕТРИКИ ДЛЯ ОЦЕНКИ
- `clv` - Значение Closing Line Value
- `clv_group` - Группа (positive/negative/neutral)
- `win_rate_by_clv` - Win Rate по группам
- `roi_by_clv` - ROI по группам

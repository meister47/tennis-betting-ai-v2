
### 🔄 Логика работы
```
1. Перед анализом матча → Проверяем кэш
2. Если матч в кэше И сегодняшний И odds не изменились >2% → Используем кэш
3. Иначе → Полный анализ → Сохраняем в кэш
4. TTL 24 часа → Автоочистка старых записей
```

## 📦 Ключевые компоненты

### 1. **AnalysisCacheManager** (`analysis-cache-manager.js`)
```javascript
// Основные методы:
- loadCache()           // Загрузка с проверкой целостности
- generateMatchId()     // Генерация уникального ID матча
- getCachedAnalysis()   // Получение кэшированного результата
- saveAnalysis()        // Сохранение результата в кэш
- cleanExpiredEntries() // Очистка старых записей (24 часа)
- getStats()           // Статистика хитов/промахов
- clearCache()         // Полная очистка кэша
```

### 2. **Структура кэша**
```json
{
  "version": "1.0",
  "created": "2026-04-16T19:30:00Z",
  "stats": {
    "totalEntries": 12,
    "hits": 8,
    "misses": 4,
    "hitRate": "66.7%"
  },
  "entries": {
    "nadal_vs_federer_atp_2026-04-16": {
      "timestamp": "2026-04-16T10:00:00Z",
      "odds_snapshot": 2.75,
      "model_prob": 0.42,
      "edge": 0.072,
      "recommendation": "bet",
      "stake": 68
    }
  }
}
```

### 3. **Интеграция в основной скрипт**
```javascript
// Добавлено в real-today-analysis-min-odds.js:

// Инициализация кэша
```

## 🛡️ Безопасность и надёжность

### ✅ Гарантии
1. **Отказоустойчивость:** При ошибке чтения кэша → работа продолжается без него
2. **Целостность:** Проверка JSON при загрузке, автосоздание при повреждении
3. **Актуальность:** TTL 24 часа + проверка изменения odds >2%
4. **Неблокирующая:** Ошибки не прерывают основной анализ

### 🔒 Защита от устаревания
- **TTL 24 часа** → гарантия актуальности
- **Проверка коэффициентов** → инвалидация при изменении >2%
- **Daily cleanup** → автоматическое удаление старых записей

## 📞 Поддержка и мониторинг

### Ключевые метрики для мониторинга
1. **Hit Rate** = hits / (hits + misses) * 100%
2. **Ускорение** = время без кэша / время с кэшом
3. **Экономия CPU** = % сокращения вычислений

### Рекомендации по использованию
```bash
# Утренний запуск (полный анализ)
node scripts/real-today-analysis-min-odds.js

# Дневные проверки (используют кэш)
node scripts/update-cache-now.js      # Обновление API кэша
node scripts/real-today-analysis-min-odds.js  # Быстрый анализ

# Управление кэшем
node scripts/analysis-cache-manager.js status  # Статистика
```
